# DarkestStatChanger — Error Code Reference

오류가 발생하면 오류 다이얼로그와 `crash_log.txt`에 `[Exxx]` 형식의 코드가 표시됩니다.
이 문서에서 코드별 원인과 해결 방법을 확인하세요.

---

## E1xx — 영웅 파일 (Hero .info.darkest)

| 코드 | 발생 조건 | 해결 방법 |
|------|-----------|-----------|
| **E101** | 영웅 `.info.darkest` 파일을 읽는 데 실패했습니다. 파일이 없거나, 다른 프로그램이 점유 중이거나, 읽기 권한이 없는 경우 발생합니다. | 파일 경로가 올바른지 확인하세요. 파일이 존재하고 다른 프로그램에서 열려있지 않은지 확인하세요. |
| **E102** | 저장 전 백업 파일(`.bak`) 생성에 실패했습니다. 디스크 공간 부족 또는 폴더 쓰기 권한 없을 때 발생합니다. | 파일이 있는 폴더에 쓰기 권한이 있는지, 디스크 여유 공간이 있는지 확인하세요. |
| **E103** | 영웅 파일 저장에 실패했습니다. 파일이 읽기 전용이거나, 다른 프로그램이 점유 중이거나, 디스크 오류가 있을 때 발생합니다. | 파일이 읽기 전용(Read-only)으로 설정되어 있지 않은지 확인하세요. 게임이 실행 중인 경우 게임을 종료 후 시도하세요. |

---

## E2xx — 영웅 파일 파싱

| 코드 | 발생 조건 | 해결 방법 |
|------|-----------|-----------|
| **E201** | `combat_skill`, `riposte_skill`, 또는 `combat_move_skill` 줄에서 `:` 구분자를 찾지 못했습니다. 파일 형식이 표준 `.darkest` 포맷을 따르지 않을 때 발생합니다. | 해당 스킬 줄이 `combat_skill: .id "..." ...` 형식을 따르는지 확인하세요. 수동으로 파일을 편집한 경우 `:` 누락 여부를 확인하세요. |

---

## E3xx — 이펙트 파일 (.effects.darkest)

| 코드 | 발생 조건 | 해결 방법 |
|------|-----------|-----------|
| **E301** | 이펙트 파일을 읽는 데 실패했습니다. 파일이 없거나, 다른 프로그램이 점유 중이거나, 읽기 권한이 없는 경우 발생합니다. | 로드된 이펙트 파일 경로가 유효한지 확인하세요. 파일이 삭제되었거나 이동된 경우 다시 Load Effects File을 클릭하세요. |
| **E302** | 이펙트 파일 저장 전 백업 파일(`.bak`) 생성에 실패했습니다. | 파일이 있는 폴더에 쓰기 권한이 있는지, 디스크 여유 공간이 있는지 확인하세요. |
| **E303** | 이펙트 파일 저장에 실패했습니다. 파일이 읽기 전용이거나 다른 프로그램이 점유 중일 때 발생합니다. | 파일이 읽기 전용으로 설정되어 있지 않은지 확인하세요. |

---

## E4xx — 캠핑 스킬 JSON

| 코드 | 발생 조건 | 해결 방법 |
|------|-----------|-----------|
| **E401** | 캠핑 스킬 JSON 파일을 읽거나 파싱하는 데 실패했습니다. JSON 파일이 손상되었거나 형식이 맞지 않을 때 발생합니다. | `*.camping_skills.json` 파일이 유효한 JSON 형식인지 확인하세요. JSON 유효성 검사 도구(예: jsonlint.com)로 파일을 검사해보세요. |
| **E402** | 캠핑 스킬 JSON 저장 전 백업 파일(`.bak`) 생성에 실패했습니다. | 파일이 있는 폴더에 쓰기 권한이 있는지, 디스크 여유 공간이 있는지 확인하세요. |
| **E403** | 캠핑 스킬 JSON 저장에 실패했습니다. | 파일이 읽기 전용으로 설정되어 있지 않은지 확인하세요. |

## E9xx — 애플리케이션 오류

| 코드 | 발생 조건 | 해결 방법 |
|------|-----------|-----------|
| **E900** | UI 스레드에서 예상치 못한 예외가 발생했습니다. 위의 E1xx~E4xx 코드에 해당하지 않는 처리되지 않은 오류입니다. | `crash_log.txt`의 내용을 확인하고 개발자에게 제보하세요. |
| **E901** | 백그라운드 스레드에서 처리되지 않은 예외가 발생했습니다. 앱이 비정상 종료될 수 있습니다. | `crash_log.txt`의 내용을 확인하고 개발자에게 제보하세요. |
| **E902** | 앱 시작 중 예외가 발생했습니다. 필요한 파일이나 라이브러리가 누락되었을 수 있습니다. | .NET Framework 4.7.2 이상이 설치되어 있는지 확인하세요. 앱 파일이 모두 정상적으로 압축 해제되었는지 확인하세요. |

```
DarkestStatChanger.DscException: [E201] Combat skill line is missing ':' separator. ...
--- Stack Trace ---
   at DarkestStatChanger.Services.DarkestFileParser.ParseCombatSkill(String line) ...
```
